

⚙️ Setup Instructions

1. Backend

cd backend
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
uvicorn main:app --reload --port 8000

2. Frontend

cd frontend
npm install
npm run dev




3 Access
	•	Frontend → http://localhost:5173
	•	Backend Docs → http://localhost:8000/docs
